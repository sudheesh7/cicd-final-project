# cicd-final-project
CI/CD Final Project for Coursera - Continuous Integration and Continuous Delivery Pipeline

## Project Name

**Project:** Continuous Integration and Continuous Delivery (CI/CD) Pipeline Implementation

## Project Details

This project demonstrates a complete CI/CD pipeline using:
- **GitHub Actions** for continuous integration automation
- **Tekton** for task management and pipeline orchestration
- **OpenShift** for container orchestration and deployment
- **Jenkins** for continuous delivery workflows

## Technologies Used

- Docker
- Kubernetes/OpenShift
- Jenkins
- GitHub Actions
- Tekton
- Node.js/JavaScript

## Course

Continuous Integration and Continuous Delivery (CI/CD) - IBM on Coursera
